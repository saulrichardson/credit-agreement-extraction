- Located the grid by searching for “Fixed Charge Coverage Ratio” in `scratch/task05/0001819974-23-000003/02_EX-10.1.txt`. The definition of “Applicable Margin” (line 10) contains the only tiered schedule, linking Term SOFR and Base Rate spreads to Tier I–III FCCR bands.

### 0001140361-23-000246 (Kimco Realty Corporation)
- The base Status grid in `scratch/task05/0001140361-23-000246/08_EX-10.1.txt:1417-1570` sets Term Benchmark/RFR/Money Market and ABR/Canadian Prime margins for Levels I–VI. Status definitions reference the S&amp;P/Moody’s thresholds described later in the same exhibit (`08_EX-10.1.txt:2828-2850`).
- Sustainability adjustments (`08_EX-10.1.txt:1490-1512`) allow the same table to apply with 1 bp lower margins whenever the borrower delivers a Sustainability Grid Notice confirming the KPI target. If the KPI beats the target by at least 1%, the margins drop by an additional 1 bp but never below zero (line 1526).
- Facility Fee Rate and the L/C Fee Rate tables appear at `08_EX-10.1.txt:2031-2090` and `08_EX-10.1.txt:2288-2403` respectively, with the L/C section also providing the sustainability version. All of these depend on the same Level I–VI status definition.

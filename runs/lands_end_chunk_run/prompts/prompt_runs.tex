This booklet captures the five ``first pass'' prompts we put in front of the Lands' End agreement along with the verbatim JSON that each model returned. Treat it like a lab notebook: every instruction the model received and every answer it produced appear back-to-back so we can review or rerun the process later without digging through scripts or command logs.
